# Production 

## Installation local 

```bash
docker compose up
```

## Accéder à l'application
ouvrez votre navigateur et accédez à l'url http://localhost:3000/accueil

## créer les données dans la bdd :
ouvrez votre navigateur et accédez à l'url http://localhost:8080
